# 🎓 Sistema Acadêmico PostgreSQL

Projeto desenvolvido para a disciplina de Banco de Dados utilizando PostgreSQL.

O sistema tem como objetivo gerenciar informações acadêmicas de:

* alunos;
* professores;
* cursos;
* disciplinas;
* turmas;
* matrículas.

---

# 🚀 Tecnologias Utilizadas

* PostgreSQL
* SQL
* pgAdmin
* BrModelo
* GitHub

# 🎯 Objetivo Acadêmico

O projeto foi desenvolvido com o objetivo de aplicar conceitos fundamentais e avançados de Banco de Dados Relacional utilizando PostgreSQL.

Durante o desenvolvimento foram utilizados:

* modelagem relacional;
* integridade referencial;
* consultas SQL;
* manipulação de dados;
* automação;
* otimização de consultas.
